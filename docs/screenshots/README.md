# Screenshots

Place demo screenshots and a short GIF in this folder. Keep file names simple and lowercase, for example:

- `upload-example.png`
- `analysis-result.png`
- `demo.gif`

When adding images to this repository, reference them using relative paths in Markdown, e.g.:

```markdown
![Upload example](docs/screenshots/upload-example.png)
```
